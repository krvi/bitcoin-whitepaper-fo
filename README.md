# Bitcoin hvítbókin á føroyskum
Hetta er ein føroysk týðing av Bitcoin hvítbókini eftir Satoshi Nakomoto.
Upprunaverkið skjalið er tøkt á [bitcoin.org](https://bitcoin.org/bitcoin.pdf).

## Rættingar
Tú er vælkomin at gera "Pull Requests" við ábótum, ella stovna eitt "Issue" við feilfráboðanum.

## Tøkk til
- [Satoshi Nakamoto](https://bitcoin.org/bitcoin.pdf) fyri upprunaverkið og uppfinningina.
- [arvicco](https://github.com/arvicco/bitcoinwhitepaper/) fyri ODF útgávuna av Bitcoin hvítbókini, ið var eitt grundarlag fyri hesum skjalinum.
- [Orðafragd](https://www.ordafragd.com/) fyri rættlestur.
- [forkh](https://github.com/forkh/) fyri sparring.
